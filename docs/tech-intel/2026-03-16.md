## Project Stack

### Context7 MCP — Critical Vulnerability Patched
**Relevance**: HIGH | **Urgency**: IMMEDIATE
**Source**: [Noma Security](https://noma.security/blog/contextcrush-context7-the-mcp-server-vulnerability/)
**Summary**: "ContextCrush" vulnerability in Context7 MCP server — custom rules could inject unescaped content enabling prompt injection. Patched February 23, disclosed March 5. Affects all users with custom rules. Update to v2.1.4.
**Action**: Verify cognitive-core's Context7 integration uses latest version. Add to security-baseline skill awareness.

### Python Security Advisories
**Relevance**: MEDIUM | **Urgency**: SOON
**Source**: [Red Hat RHSA-2026:2128](https://access.redhat.com/errata/RHSA-2026:2128)
**Summary**: CPython wsgiref.headers.Headers allows header newline injection (CVE-2026-0865). IMAP/POP3 command injection (CVE-2025-15366/15367). Email header injection (CVE-2026-1299). RHEL 8 patches available.
**Action**: Not directly affecting cognitive-core (Bash-primary), but relevant for Python adapter users. Note for tech-intel awareness.
